# cgv-data

Shared data repository for CGV projects.

This repo contains stable data consumed by CGV apps and tools.

## Includes

- Bible texts
- Lexicon observation data
- Interlinear data
- Shared schemas

## Rule

Engines generate data.
Apps consume data.
Shared published data lives here.

## License note

Only redistributable data should be committed.

Licensed Bible texts must remain local and gitignored.
